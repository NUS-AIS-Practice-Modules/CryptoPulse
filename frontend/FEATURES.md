# Frontend 功能清单

## 规则

- 一次只做一个功能
- 验证通过才算完成
- 优先完成聊天主流程

## 功能列表

### FE-001: 项目初始化（priority: 1）

**描述**：初始化 React + TypeScript + Tailwind + Vite 项目。

**验证标准**

- [ ] `npm install` 成功
- [ ] `npm run dev` 可启动
- [ ] `npm run build` 成功
- [ ] Tailwind CSS 生效

### FE-002: 聊天页面 UI（priority: 2）

**描述**：实现聊天主页面。

**验证标准**

- [ ] 输入框可输入内容
- [ ] 点击发送按钮可提交
- [ ] 用户消息显示右侧
- [ ] AI 消息显示左侧
- [ ] 消息列表支持滚动
- [ ] 页面布局正常

### FE-003: 接入聊天 API（priority: 3）

**描述**：连接 `POST /api/chat`。

**验证标准**

- [ ] 点击发送调用 API
- [ ] 返回 `reply` 正常显示
- [ ] loading 动画正常
- [ ] `conversation_id` 支持多轮对话
- [ ] API 失败时有错误提示

### FE-004: 文件上传组件（priority: 4）

**描述**：实现文件上传功能 UI。

**验证标准**

- [ ] 可选择文件
- [ ] 显示文件名
- [ ] 支持 PDF/TXT 文件
- [ ] 上传区域 UI 正常显示

### FE-005: Dashboard 页面（priority: 5）

**描述**：实现数据可视化页面。

**验证标准**

- [ ] 页面可访问
- [ ] 折线图展示趋势
- [ ] 饼图展示情绪占比
- [ ] 数据来自 API
- [ ] 筛选器可切换时间范围

### FE-006: 状态检测与错误处理（priority: 6）

**描述**：处理系统状态与异常提示。

**验证标准**

- [ ] `GET /api/health` 可调用
- [ ] 后端异常时提示用户
- [ ] Loading 状态正常
- [ ] 网络错误有提示信息
