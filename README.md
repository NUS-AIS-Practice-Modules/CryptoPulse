# CryptoPulse

CryptoPulse is a multi-module team project for cryptocurrency sentiment analysis, retrieval-augmented question answering, and a web chat interface.

## Modules

- `frontend/`: React web application for chat and dashboard UX
- `chatbot/`: FastAPI integration layer and public REST API
- `lora/`: LoRA fine-tuning and inference module
- `rag/`: Retrieval and context-building module
- `shared/`: shared Python types used across backend-facing modules

## Start Here

1. Read `AGENTS.md`
2. Read `ARCHITECTURE.md`
3. Read `docs/INTERFACES.md`
4. Run `./init.sh`
